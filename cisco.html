Basic Switch Configuration

A console cable has two parts RJ45 to RS232 and RS232 to USB.When accessing a switch with a console cable, the RJ45 connector of the console cable should be connected to the console port of the switch and the USB connector should be connected to the USB port of the laptop.
Then Go to the Computer management (Windows + X) then Device Manager > Ports (COM & LPT) > Prolife USB to Serial COM port (COM1 or COM2 or else)
Now From Putty or Tera Team software we can access in the switch CLI(Command Line Interface) by selecting serial option and Input the COM Number.

In the switch CLI(Command Line Interface) by default it is in USER Mode. In the USER Mode we can apply some show command by using this symbol ( ? )

By using enable (en) command from this user mode we can go to privilege mode and (#) is the symbol that you can understand it is in the Previladge Mode or Not. In this privilege mode the output of everything can be shown, then if we want to configure something, We can go to global configuration mode [Global Configuration mode, mode allows users to modify the running system configuration.], for this we have to use configuration terminal (Conf t) command in privilege mode
